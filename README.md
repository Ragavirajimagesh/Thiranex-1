# Thiranex-1
Data Analytics
📊 Sales and Revenue Analysis – E-commerce Dashboard
📌 Overview

This project presents an interactive Power BI dashboard developed for Thiranex, designed to analyze sales performance, revenue, and profitability in an e-commerce environment. It provides key insights into product categories, regions, and sales trends to support data-driven decision-making.

🏢 Company

Thiranex

🎯 Objectives
Track overall sales, profit, and quantity
Analyze performance by product category
Identify top-performing products
Understand regional sales distribution
Provide interactive filtering for better insights
📈 Key Metrics
💰 Total Sales: 11M
📦 Total Quantity Sold: 17K
📊 Total Profit: 1.84M
📊 Dashboard Features
1. KPI Cards
Displays high-level metrics:
Total Sales
Total Quantity
Total Profit
2. Profit by Category
Bar chart showing profit contribution from:
Electronics
Accessories
Office Supplies
3. Sales by Product
Pie chart visualizing sales distribution across different products
4. Category-wise Comparison
Combined chart showing:
Profit
Sales trends across categories
5. Regional Analysis
Map visualization displaying sales distribution across regions:
North America
Europe
Asia
Africa
6. Interactive Filters
Filter data using:
Category
Order Date
🛠️ Tools & Technologies
Power BI – Data visualization and dashboard creation
Power Query – Data cleaning and transformation
DAX – Calculated measures and analysis
📂 Dataset

The dataset includes:

Order Date
Product Name
Category
Sales
Quantity
Profit
Region
🚀 How to Use
Download the .pbix file from this repository
Open it in Power BI Desktop
Use slicers (filters) to explore data dynamically
Hover over visuals to see detailed insights
💡 Insights Gained
Electronics category generates the highest profit
Accessories show consistent performance
Office supplies have comparatively lower contribution
Products like Monitors contribute significantly to revenue
